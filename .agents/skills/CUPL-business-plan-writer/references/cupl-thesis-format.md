# 中国政法大学本科毕业论文撰写格式规范

> 此文件供 Step 5 撰写时参考。当 Step 0 检测到项目来自中国政法大学或用户提供了法大格式规范文件时使用。

---

## 核心格式要求速查

### 页面设置
- A4 纸（210mm × 297mm），纵向，单面打印
- 页边距：上下 2.5cm，左右 2.0cm

### 页眉页脚
- 页眉（除封面外所有页）：**中 国 政 法 大 学 本 科 生 毕 业 论 文**，小四号宋体，居中，每字间空一格
- 页脚：Times New Roman 五号，居中，仅页码
- 摘要→目录：罗马数字页码 I, II, III...
- 引言→结尾：阿拉伯数字页码 1, 2, 3...
- 页码数字两侧不加修饰线

### 标题体系：汉字层级标题制

法大本科论文**不使用阿拉伯数字层级标题制**（如 1、1.1、1.1.1），采用汉字层级：

```
一、（一级标题）— 小三号加粗宋体，居中，单倍行距，段前段后各 24 磅
（一）（二级标题）— 四号加粗宋体，左端顶格，固定行距 20 磅，段前段后各 18 磅
1.（三级标题）— 四号宋体，左侧缩进 0.85cm，固定行距 20 磅，段前 12 磅段后 6 磅
（1）（四级标题）— 四号楷体，左侧缩进 0.85cm，固定行距 20 磅，段前段后各 6 磅
```

**重要：** 引言、结论与一级标题格式相同，二字之间空两格编排。

### 正文格式
- 字体：四号宋体（英文用 Times New Roman）
- 行距：固定值 20 磅
- 每段首行缩进 2 汉字符，回行顶格

### 引文段落
- 引用过长时：另起段落，首行缩进 2 字符，左侧缩进 2 字符
- 字体：四号楷体

### 中文摘要
- 题目：三号黑体，居中，单倍行距，段前段后各 24 磅
- 【摘要】：四号加粗黑体，顶格，单倍行距，段前段后各 18 磅
- 内容：小四号宋体，固定行距 20 磅
- 【关键词】：四号黑体，顶格，单倍行距，段前段后 18 磅
- 关键词：小四号宋体，之间空两格，3-5 个

### 英文摘要
- 题目：三号 Times New Roman 加粗居中，全部大写
- 【ABSTRACT】：四号 Times New Roman 加粗，顶格
- 内容：小四号 Times New Roman，固定行距 20 磅
- 【KEYWORDS】：四号 Times New Roman 加粗
- 关键词：小写，空格分隔

### 目录
- 【目录】：三号黑体，居中，单倍行距，段前段后各 24 磅
- 内容：小四号宋体，固定行距 20 磅
- 一级标题居左，二级缩进 1 字符，三级缩进 2 字符

### 脚注格式（法大专有）
- 正文中注释用带圈阿拉伯数字上标标注，**每页重新编号**
- 脚注内容：宋体小五号，两端对齐，单倍行距
- 序号与脚注内容之间空半个汉字符

**中文独立文献注释（首次引用）：**
```
著者姓名、文献名、卷册序号、出版单位、出版时间、页码。
```

**再次引用同一文献：**
```
只需注出作者姓名、著作名和资料所在页码。
```

**期刊文章注释：**
```
作者：《文章名》，载《期刊名》年份及期数。
```

**网络资料注释：**
```
必须注明网址和最后访问时间。
```

### 参考文献
- 另起页编排
- 标题：小三号加粗宋体，一级标题格式
- 正文：五号字，汉字宋体，英文 Times New Roman，固定行距 16 磅
- 可分类著录：一、著作类 二、期刊论文类 三、文集论文类 四、学位论文类 五、网址及其他类
- 每类内用 [1]、[2]... 编号

### 图表
- 图序/图名：宋体五号，居中，单倍行距，段前 6 磅段后 12 磅
- 表序/表名：宋体五号，居中，单倍行距，段前 6 磅段后 12 磅
- 编号连续至附录前，与章节编号无关

### 装订顺序
封面 → 摘要及关键词 → 目录 → 正文 → 参考文献 → 附录 → 致谢

## 注释格式速查

### 著作类
```
作者（或主编、编）：《著作名称》（版次），出版者及出版年份，页码。
```

### 期刊类
```
作者：《文章名》，载《期刊名》年份及期数。
```

### 网页类
```
作者：《文章名》，载网站某年某月某日。最后访问时间：YYYY-MM-DD。
```

### 英文期刊类
```
Author, Title of the Article, Volume Journal Name, Page(s) (Year).
```

### 英文书籍类
```
Author, Title of the Book, Publisher, Year, Page(s).
```
