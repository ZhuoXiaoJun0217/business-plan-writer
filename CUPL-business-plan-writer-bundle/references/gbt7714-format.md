# GB/T 7714-2015 参考文献格式参考

> 此文件提供 GB/T 7714-2015 的详细格式说明和完整示例。Step 5 撰写时用作引用格式速查。

---

## 正文引用格式

本 Skill 默认使用**顺序编码制**（上标序号¹²³），在正文引用处标注上标序号，文末参考文献按出现顺序排列。

## 参考文献列表格式

### 期刊论文 [J]
```
[序号] 作者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
```

### 专著 [M]
```
[序号] 作者. 书名[M]. 版本(第一版不标注). 出版地: 出版社, 出版年.
```

### 报告 [R]
```
[序号] 作者. 题名[R]. 出版地: 出版者, 出版年.
```

### 电子资源 [EB/OL]
```
[序号] 作者. 题名[EB/OL]. (发布日期)[引用日期]. 获取和访问路径.
```

### 学位论文 [D]
```
[序号] 作者. 题名[D]. 保存地: 保存者, 年份.
```

### 会议论文 [C]
```
[序号] 作者. 题名[C]//会议名称. 会议地点, 会议日期.
```

### 专利 [P]
```
[序号] 专利所有者. 专利名: 专利号[P]. 公告日期.
```

### 标准 [S]
```
[序号] 标准制定机构. 标准编号 标准名称[S]. 出版地: 出版社, 出版年.
```

## 完整示例

```
[1] 国家统计局. 中国统计年鉴 2025[M]. 北京: 中国统计出版社, 2025.
[2] 张明, 李华. 人工智能创业生态系统研究[J]. 科学学研究, 2025, 43(2): 201-210.
[3] 国务院. 关于促进创业投资持续健康发展的若干意见[EB/OL]. (2024-09-20)[2026-05-26]. http://www.gov.cn/xxx.
[4] 王磊. 共享经济商业模式创新研究[D]. 北京: 清华大学, 2025.
[5] 艾瑞咨询. 2025年中国二手电商行业研究报告[R]. 上海: 艾瑞咨询, 2025.
[6] 教育部. 关于深化高等学校创新创业教育改革的实施意见[EB/OL]. (2023-06-10)[2026-05-26]. http://www.moe.gov.cn/xxx.
```

## 常见问题

- **3 位以内作者全部列出**，超过 3 位只列前 3 位加 ",等" 或 ",et al"
- **无作者**：以题名开头
- **无出版日期**：标注 [出版年不详]
- **中文文献用中文标点**（. ，）：英文文献用英文标点（. ,）
- **[EB/OL] 必须同时标注发布日期和引用日期**
